## Local usage

```
Prerequisites:

node >= 18.7.0
npm >= 8.15.0
```

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start local server
   ```js
   npm run dev
   ```
